# 牌宗 · 五宗归一（长线桌面版）

基于《牌宗_满分版玩法设计方案》的完整可玩 HTML5 游戏，目标支撑 **约 50 小时** 复玩深度。

纯静态站点（HTML/CSS/JS + 素材），存档在浏览器 `localStorage`，适合 **GitHub Pages** 托管。

## 本地启动

浏览器打开 `index.html`，或：

```bash
# Python
python -m http.server 8765

# 或 Node
npx serve .
```

访问：`http://127.0.0.1:8765/`

## 部署到 GitHub Pages

### 1. 创建仓库并上传

在 GitHub 新建空仓库（例如 `paizhong`），**不要**勾选自动添加 README（本地已有）。

在本项目目录执行（把 `你的用户名` 换成自己的）：

```bash
git init
git add .
git commit -m "feat: 牌宗静态站，可 GitHub Pages 部署"
git branch -M main
git remote add origin https://github.com/你的用户名/paizhong.git
git push -u origin main
```

### 2. 打开 Pages

1. 打开仓库 → **Settings** → **Pages**
2. **Source** 选 **Deploy from a branch**
3. Branch 选 **`main`**，文件夹选 **`/ (root)`**
4. 点 **Save**

约 1～2 分钟后访问：

```text
https://你的用户名.github.io/paizhong/
```

> 仓库名若是 `你的用户名.github.io`，则根地址为 `https://你的用户名.github.io/`。

### 3. 验证

- 首页能打开，CSS/图片不 404  
- 能开始论道；刷新后进度还在（同浏览器）  
- 若样式旧：`Ctrl+F5` 强刷  

### 4. 以后更新

改完代码后：

```bash
git add .
git commit -m "更新说明"
git push
```

Pages 会自动重新发布。

## 核心玩法

- 压牌合法性 ≈ 斗地主  
- 得分：`牌理 × 气 × 宗势`，达门槛即通关  
- 操作：出牌 / 过牌 / 机关  
- 构筑：奇牌被动 + 心法 + 锦囊

## 建议体验路径（50h）

1. **0–2h** 常道熟悉压牌与爆分，通第一～三境  
2. **2–8h** 解锁牌客与卡池，尝试 4 种流派  
3. **8–20h** 险途/宗师重刷八境，冲成就与图鉴  
4. **20–50h** 传说难度 + 无尽冲层 + 每日 + 全收集  

进度保存在浏览器 `localStorage`（`paizong_meta_v2`）。
